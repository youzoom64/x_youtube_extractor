⏰ スケジューラー版の使い方

基本コマンド

# 設定ファイルの確認 python scheduler/auto_scheduler.py --list

# 特定ジョブのテスト実行
python scheduler/auto_scheduler.py --test "朝のニュース分析"

# スケジューラー本稼働
python scheduler/auto_scheduler.py

# バックグラウンド実行（Linux/Mac）
nohup python scheduler/auto_scheduler.py > scheduler.log 2>&1 &

設定ファイル (scheduler_config.json)

{
"schedules": [
{
"name": "朝のニュース分析",
"query": "ニュース OR 時事 OR 最新情報",
"time": "08:00",
"count": 30,
"format": "txt",
"sort_type": "latest",
"claude_analysis": true,
"analysis_prompt": "重要なニュースを3つ選んで要約してください",
"days": ["monday", "tuesday", "wednesday", "thursday", "friday"]
}
]
}

設定項目の説明

項目 説明 例
name ジョブ名 "朝のニュース分析"
query 検索クエリ "AI OR 人工知能"
time 実行時刻 "08:00"
count 取得件数 30
format 出力形式 "txt" / "json"
sort_type ソート順 "latest" / "top"
claude_analysis Claude分析 true / false
days 実行曜日 ["daily"] または ["monday", "friday"]

🔧 トラブルシューティング

Chrome接続エラー

症状: 🔴 Chrome接続エラー
解決方法:
# 1. Chromeを完全終了
taskkill /f /im chrome.exe

# 2. デバッグモードで再起動
chrome --remote-debugging-port=9222

# 3. Claude.aiに再ログイン

ツイート取得失敗

症状: ❌ ツイートが取得できませんでした
確認事項:
  • ネットワーク接続
  • Twitterにログインしているか
  • 検索クエリが正しいか
  • 非公開アカウントでないか

ファイルアップロード失敗

症状: ❌ ファイルアップロードエラー
Claude.aiページを手動で確認
ページをリロード
新しいチャットを作成
再実行

Python ModuleNotFoundError

症状: ModuleNotFoundError: No module named 'XXX'
# 仮想環境を確認 which python pip list # 不足モジュールをインストール pip install 不足しているモジュール名
